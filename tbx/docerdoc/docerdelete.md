# docerdelete :recycle:

Delete Doc'er artifacts

## Syntax

`docer delete f` deletes Doc'er artifacts in the folder f:
* HTML files with corresponding Markdown files
* PNG files with corresponding MATLAB scripts
* cascading stylesheets and JavaScript scripts
* index files `info.xml` and `helptoc.xml` and folder `helpsearch-v4`
